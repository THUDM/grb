Install and Setup
=================

System requirements
-------------------

GRB works with Ubuntu 16.04 or higher and requires Python version 3.6 or higher.

Install via Pip
---------------

We recommend installing GRB using ``pip``.

.. code:: bash

   pip install grb

Install from source
-------------------

.. code:: bash

   git clone git@github.com:THUDM/grb.git
   cd grb
   pip install -e .
